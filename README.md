# pLCA-FR-H2-production

This repository contains the code and data that has been used for the article 'Environmental impacts of the French Hydrogen Strategy: a prospective Life Cycle Assessment'. Authors: François-Xavier Dézert and Guillaume Batôt. 

The code produces prospective Life Cycle Inventory databases of French Hydrogen Production and Electricity Mixes. 
Prospective electricity mixes are modelled for the N03, N1, N2, M23, and M0 reference scenarios of the RTE report "Futurs Energétiques 2050", published in 2022 (https://www.rte-france.com/donnees-publications/etudes-prospectives/futurs-energetique-2050#Lesresultatsdeletude).
Data for prospective hydrogen production is derived from the IEA database and an exhaustive internet review (ended in october 2025). More details on the data are provided in the article.
